# House Price Prediction Using Supervised Learning

## Overview
This project predicts house prices using the California Housing dataset with supervised learning techniques. It implements Linear Regression and Random Forest models to achieve an RÂ² score of 0.85 with Random Forest, demonstrating robust regression performance.

## Dataset
- **Source**: California Housing dataset from scikit-learn
- **Features**: Median income, house age, average rooms, etc.
- **Target**: Median house price
